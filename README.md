# Learning-GSAP

A collection of GSAP (GreenSock Animation Platform) demo projects and experiments showcasing various animations, techniques, and interactive effects.

## About

This repository is meant for learning and experimenting with GSAP animations â€” from simple text effects to scroll-based animations, interactive components, landing pages, etc.  

It helps to:

- See different animation patterns

- Use GSAP plugins, e.g. ScrollTrigger, Draggable, etc.  
- Use code as a reference or starting point for your own animations
